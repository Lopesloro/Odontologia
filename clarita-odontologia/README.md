# Clarità Odontologia - Landing Page

Landing page moderna, elegante e de alta conversão para a clínica odontológica Clarità Odontologia.

## 🎨 Design

- **Estilo**: Clean, minimalista e sofisticado
- **Abordagem**: Mobile-first e responsivo
- **Foco**: Alta conversão com muito whitespace e tipografia elegante

## 📁 Estrutura de Pastas

```
clarita-odontologia/
│
├── index.html              # Página principal
├── css/
│   └── style.css          # Estilos principais
├── js/
│   └── script.js          # Funcionalidades interativas
├── assets/
│   └── images/            # Imagens da clínica
└── README.md              # Este arquivo
```

## 🚀 Como Usar

1. **Localmente**: Abra o arquivo `index.html` em qualquer navegador
2. **Hospedagem**: Faça upload de todos os arquivos para seu servidor web

## 📝 Seções Incluídas

1. **Header** - Logo, navegação e botão CTA
2. **Hero** - Título principal com ondas sutis de fundo
3. **Especialidades** - Cards dos serviços oferecidos
4. **Corpo Clínico** - Apresentação das dentistas
5. **Footer** - Contato completo com design azul

## 🎯 Funcionalidades

- ✅ Menu mobile com animação suave
- ✅ Navegação suave entre seções
- ✅ Header com efeito glassmorphism no scroll
- ✅ Botão flutuante do WhatsApp
- ✅ Scroll reveal animation
- ✅ Efeito parallax nas ondas de fundo
- ✅ Botão voltar ao topo
- ✅ Máscara automática para telefone
- ✅ 100% responsivo

## 🔧 Personalização

Para alterar as cores, edite as variáveis CSS no arquivo `css/style.css`:

```css
:root {
    --color-primary: #889EBD;        /* Azul principal */
    --color-text-primary: #4A5248;   /* Textos */
    /* ... */
}
```
